# commons.engineering

Hub site for Commons Engineering — a discipline for designing living systems toward vitality rather than extraction.

## Structure

```
content/
  manifests/    Founding documents
  engines/      The four engine descriptions
  about/        About Commons Engineering
docs/
  adr/          Architecture Decision Records
static/         Static assets
layouts/        Hugo templates
```

## Engines

| Engine | Repository | Domain |
|---|---|---|
| Pattern Engine | [pattern-engine](https://github.com/Commons-Engineering/pattern-engine) | patterns.commons.engineering |
| Lighthouse Engine | [lighthouse-engine](https://github.com/Commons-Engineering/lighthouse-engine) | lighthouses.commons.engineering |
| Context Engine | [context-engine](https://github.com/Commons-Engineering/context-engine) | — |
| Community Engine | [community-engine](https://github.com/Commons-Engineering/community-engine) | — |

## Tech Stack

- **Hugo** — static site generator
- **Cloudflare Pages** — hosting and CDN
- **Pagefind** — client-side search
